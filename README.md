# Intro to Web Development

Welcome to this comprehensive introduction to web development! This course will teach you how to build interactive, dynamic, and visually appealing websites using HTML, CSS, and JavaScript.

## Course Structure

This course is divided into three main parts, each focusing on a core web technology:

### [Part 1: HTML](./01-html/README.md)
Learn the foundation of web pages - HTML (HyperText Markup Language)
- [HTML Basics](./01-html/01-basics/README.md)
- [HTML Forms and Input](./01-html/02-forms/README.md)
- [HTML Semantic Elements](./01-html/03-semantic/README.md)

### [Part 2: CSS](./02-css/README.md)
Master styling and layout with CSS (Cascading Style Sheets)
- [CSS Basics](./02-css/01-basics/README.md)
- [CSS Layout](./02-css/02-layout/README.md)
- [CSS Styling and Design](./02-css/03-styling/README.md)

### [Part 3: JavaScript](./03-javascript/README.md)
Add interactivity and dynamic behavior with JavaScript
- [JavaScript Basics](./03-javascript/01-basics/README.md)
- [DOM Manipulation](./03-javascript/02-dom/README.md)
- [Events and Interactivity](./03-javascript/03-events/README.md)

### [Final Project](./final-project/README.md)
Put it all together and build your own website!

## How to Use This Course

1. Start with Part 1 (HTML) and work through each subpart in order
2. Read the lesson material in each README
3. Complete the three practice problems for each subpart
4. Move on to the next subpart once you're comfortable
5. After completing all three parts, tackle the final project

## Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- A text editor (VS Code, Sublime Text, or any code editor)
- Enthusiasm to learn!

Navigate to [Part 1: HTML Basics](./01-html/01-basics/README.md) to begin your journey!
