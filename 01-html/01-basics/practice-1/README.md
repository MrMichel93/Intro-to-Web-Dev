# Practice 1: My First Web Page (Easy)

## Objective

Create your first complete HTML page with basic elements including headings, paragraphs, and a list.

## Requirements

Create a file named `index.html` that includes:

1. A proper HTML5 document structure (DOCTYPE, html, head, body)
2. A page title in the `<title>` tag: "My First Web Page"
3. An `<h1>` heading with the text "Welcome to My First Web Page"
4. An `<h2>` heading with the text "About Me"
5. A paragraph introducing yourself (minimum 2 sentences)
6. An `<h2>` heading with the text "My Hobbies"
7. An unordered list with at least 3 hobbies

## Example Output

Your page should look similar to this:

```
Welcome to My First Web Page
=========================

About Me
--------
Hello! My name is Alex and I'm learning web development. I'm excited to build amazing websites.

My Hobbies
----------
• Reading books
• Playing guitar
• Hiking
```

## Tips

- Remember to include the `<!DOCTYPE html>` declaration at the top
- Don't forget to close all your tags
- Use proper indentation to make your code readable
- Test your page by opening it in a web browser

## Solution

Try to complete this on your own first! If you get stuck, you can check the solution file.

<summary>Click to see solution</summary>

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Web Page</title>
</head>
<body>
    <h1>Welcome to My First Web Page</h1>
    
    <h2>About Me</h2>
    <p>Hello! My name is Alex and I'm learning web development. I'm excited to build amazing websites and create interactive experiences for users.</p>
    
    <h2>My Hobbies</h2>
    <ul>
        <li>Reading books</li>
        <li>Playing guitar</li>
        <li>Hiking</li>
    </ul>
</body>
</html>
```
