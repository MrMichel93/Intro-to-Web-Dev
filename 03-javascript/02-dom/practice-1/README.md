# Practice 1: Content Manipulator (Easy)

## Objective
Practice selecting elements and modifying their content using DOM manipulation.

## Requirements

Create `index.html` and `script.js`:

1. HTML with:
   - A heading with id "title"
   - A paragraph with id "description"
   - A button "Change Content"
   - An image with id "photo"
   - A div with class "message" (initially empty)

2. JavaScript that:
   - Changes the heading text when button is clicked
   - Changes the paragraph text
   - Changes the image source
   - Adds a success message to the message div
   - Adds a CSS class to style the message

## Tips
- Use `getElementById()` to select elements
- Use `textContent` or `innerHTML` to change text
- Use `setAttribute()` for the image
- Use `classList.add()` to add classes
