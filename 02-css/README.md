# Part 2: CSS (Cascading Style Sheets)

## Introduction

CSS (Cascading Style Sheets) is the language used to style and layout web pages. While HTML provides structure, CSS brings your pages to life with colors, fonts, spacing, layouts, and animations. Think of CSS as the paint, wallpaper, and furniture that makes a house beautiful.

## What You'll Learn

In this section, you'll master:
- Selecting and styling HTML elements
- Creating responsive layouts with Flexbox and Grid
- Designing beautiful, modern interfaces
- Adding visual effects and animations
- Building mobile-friendly, accessible websites

## Subparts

1. **[CSS Basics](./01-basics/README.md)** - Learn CSS syntax, selectors, and fundamental properties
2. **[CSS Layout](./02-layout/README.md)** - Master Flexbox, Grid, and responsive design
3. **[CSS Styling and Design](./03-styling/README.md)** - Create beautiful designs with advanced styling techniques

## Prerequisites

Before starting this section, make sure you've completed [Part 1: HTML](../01-html/README.md) as you'll need to understand HTML structure to apply CSS effectively.
